# 🚀 Real-Time Crypto Analytics Dashboard

A real-time data analytics dashboard that processes live cryptocurrency data and generates actionable insights using statistical and technical analysis techniques.

---

## 📌 Project Overview

This project demonstrates how real-time data can be collected, processed, and analyzed to extract meaningful insights. The dashboard fetches live cryptocurrency data from CoinGecko and applies analytical techniques such as trend analysis, volatility measurement, and technical indicators.

---

## 🔗 Live Demo

👉 https://crypto-analytics-dashboard-daaq6hveb7o2okb4kvtkjb.streamlit.app/

---

## ⚙️ Features

* 📡 Real-time data ingestion using APIs
* 📊 Interactive dashboard using Streamlit
* 📈 Technical indicators:

  * Moving Averages (MA 7, MA 14)
  * RSI (Relative Strength Index)
  * MACD (Moving Average Convergence Divergence)
* 📉 Volatility & returns analysis
* 📢 Trading signal generation (Bullish / Bearish)
* 💼 Portfolio tracking (basic)
* 🔄 Auto-refresh for live updates

---

## 🧠 Data Analytics Techniques Used

* Data Cleaning & Structuring (time-series formatting)
* Feature Engineering (returns, moving averages)
* Statistical Analysis (volatility calculation)
* Technical Analysis (RSI, MACD)
* Insight Generation (rule-based signals)
